特殊情况1：

```
console.log(b); // function
var b = 9;
console.log(b); // 9
function b() {
}
```
解析：在创建全局上下文时由于函数名优先级高于变量名，所以在创建上下文的时候 b 为function，然后在执行时b被赋值9，所以第二次打印的b就是9
